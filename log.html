Changelog

Pembaruan terbaru dan peningkatan sistem pada antarmuka portofolio ArdyOS.

v8.1.0

Agustus 2024
  • New
    Natural Language Processing (NLP). Terminal sekarang dapat merespons perintah menggunakan bahasa sehari-hari seperti "halo", "siapa kamu", atau "lihat proyek" tanpa harus menggunakan perintah baku.
  • New
    Quick Command Chips. Menambahkan tombol perintah cepat di atas area input untuk navigasi instan bagi pengunjung non-teknis.
  • New
    Rich UI Cards. Menambahkan perintah projects yang menampilkan antarmuka kartu modern ala Bento Grid langsung di dalam terminal.
  • Improvement
    Idle Assistant. Menambahkan asisten AI yang akan memberikan panduan secara otomatis apabila pengunjung diam (idle) selama lebih dari 12 detik.

v8.0.5

Juli 2024
  • New
    ASCII Weather Art. Perintah weather sekarang menampilkan ilustrasi ASCII grafis sesuai dengan kondisi cuaca (Cerah, Hujan, Badai, dll) selain efek latar belakang.
  • New
    Dynamic Text Zoom. Menambahkan kontrol (+/-) di toolbar atas untuk memperbesar dan memperkecil ukuran font terminal secara dinamis.
  • Fix
    Autocomplete Position. Memperbaiki bug kalkulasi posisi kursor bayangan (hint) saat fitur autocomplete digunakan agar sejajar dengan ketikan pengguna.
  • Improvement
    Code Optimization. Menghapus dead code pada proses rendering teks untuk meningkatkan frame rate (FPS) terminal saat pengguna mengetik cepat.

v8.0.0

Juni 2024
  • New
    Initial Release: ArdyOS Terminal. Meluncurkan antarmuka portofolio berbasis Command Line Interface (CLI) dengan estetika brutalist-fungsional macOS.
  • New
    Live Weather API. Integrasi sistem cuaca real-time berdasarkan geolokasi pengunjung atau IP Fallback.
  • New
    Built-in ATS Resume Generator. Fitur hidden untuk mengkompilasi dan mengunduh CV berformat PDF yang ramah ATS langsung dari baris perintah.
